2. ¿Qué sabemos de las rocas?

Pensando un poco...


      Antes de comenzar a explorar el fascinante mundo de las rocas, vamos a hacer un breve repaso para ver cuánto sabemos ya sobre ellas. Piensen en lo que conocen: ¿cómo son? ¿por qué son importantes para nosotros y para el planeta? ¿Cuántos tipos conocen?

      Para hacerlo divertido, vamos a utilizar una herramienta llamada Plickers. Es un juego en el que podrán responder a preguntas que irán apareciendo en la pantalla de clase. Cada uno de ustedes tendrá una tarjeta especial, con un código único, que les permitirá escoger su respuesta sin tener que usar un dispositivo.

¿Cómo se juega a Plickers?

     


Cada estudiante recibe una tarjeta Plickers. Cada tarjeta tiene cuatro lados, cada uno representando una letra diferente (A, B, C o D).
Responde levantando tu tarjeta. Cuando haga una pregunta, verán varias opciones en la pantalla. Ustedes solo tienen que girar su tarjeta de forma que la letra que representa su respuesta (A, B, C o D) quede en la parte superior.
El profesor escanea las tarjetas. Con una aplicación especial, escanearé todas las respuestas con mi dispositivo. ¡Así veremos quién acertó y podremos ver el resultado de todos en la pantalla!
Con este juego, vamos a recordar y descubrir juntos datos interesantes sobre las plantas, y a ver qué tanto ya sabemos antes de comenzar a construir nuestro herbario digital. ¡Manos a las tarjetas y a disfrutar aprendiendo sobre el mundo vegetal!
